 # MultiSpline 0.1.1

* Added method references to DESCRIPTION in the required CRAN format
  (Hastie & Tibshirani, 1990; Bates et al., 2015; Wood, 2017)
* Replaced `\dontrun{}` with `\donttest{}` and fully runnable examples
  using built-in data (mtcars) in response to CRAN reviewer feedback

# MultiSpline 0.1.0

* Initial release on CRAN
* `nl_fit()` for fitting nonlinear single-level and multilevel models
* `nl_summary()` for structured coefficient output
* `nl_predict()` for predictions with uncertainty intervals
* `nl_plot()` for visualization of nonlinear effects
* `nl_icc()` for intraclass correlation estimation
